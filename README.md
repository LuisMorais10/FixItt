# 🚀 FixItt

Plataforma web para conectar usuários a serviços de manutenção e soluções domésticas.

---

## 📌 Sobre o projeto
O FixItt é uma aplicação fullstack desenvolvida com Django e React, com foco em resolver problemas reais de conexão entre clientes e prestadores de serviço.

---

## 🧠 Tecnologias utilizadas
- Python
- Django
- React
- PostgreSQL
- Docker
- REST APIs

---

## ⚙️ Funcionalidades
- Cadastro e autenticação de usuários
- Listagem de serviços
- Integração frontend/backend via API REST
- Gerenciamento de dados

---

## 🏗️ Arquitetura
- Backend: Django (API REST)
- Frontend: React
- Comunicação via JSON
